Technologies Used

* Ruby
* Rspec

How to use

1) Fork this repo
2) Clone this repo
3) Open a repl
4) Require lib/bank.rb
5) Initialise with Bank.new
6) You can then run the methods from there
7) e.g. bank = Bank.new
        bank.deposit_money(10)
        bank.withdraw_money(5)
        bank.bank_statement
